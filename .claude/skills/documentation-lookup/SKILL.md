# Documentation Lookup

When the user asks about libraries, frameworks, or APIs, fetch current documentation via web research.

## When to use
- Setup or configuration questions
- Code examples for a library
- API reference info
- Framework-specific patterns (React, Vite, Tailwind, Zustand, Vitest)

## Workflow
1. Identify the library/framework and version
2. Search official documentation sources
3. Return current API signatures and code examples

## Project libraries to reference
- React 19, Vite 8, Tailwind CSS v4, Zustand 5, Vitest 4
