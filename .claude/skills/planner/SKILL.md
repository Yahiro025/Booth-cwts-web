# Planner Agent

## When to Activate
- Complex multi-step features (new game, new level, physics changes)
- Refactoring across multiple files
- Adding new game type to registry

## Workflow
1. Read relevant files, understand architecture
2. Identify dependencies and ordering
3. Create ordered implementation plan with file paths
4. Identify risks and mitigations
5. Execute plan step by step

## Output
Ordered steps, files to modify, dependencies, risk assessment.
